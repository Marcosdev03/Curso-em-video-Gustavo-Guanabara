# Curso em Vídeo - Python

## Descrição do Projeto

O **Curso em Vídeo - Python** é um repositório criado para acompanhar e praticar os exercícios do curso de Python,
oferecido pelo Professor Gustavo Guanabara. Como estudante de Engenharia de Software, 
estou dedicando tempo para aprimorar meus conhecimentos em programação, 
e este repositório documenta meu progresso em Python ao longo de quatro meses de estudo.

Este projeto tem como objetivo solidificar conceitos fundamentais de programação, incluindo:

- **Introdução ao Python**: Configuração do ambiente, sintaxe básica e execução de programas.
- **Estruturas de Controle**: Condicionais, loops e como controlar o fluxo do programa.
- **Funções**: Definição e chamada de funções para organizar o código.
- **Listas e Tuplas**: Manipulação de coleções de dados.
- **Dicionários**: Uso de estruturas de dados para armazenar pares de chave-valor.
- **Tratamento de Erros**: Como lidar com exceções e garantir que o programa continue funcionando.

Através deste repositório, espero não apenas praticar a resolução de problemas, mas também compartilhar meu aprendizado com outros iniciantes que desejam seguir o mesmo caminho.
